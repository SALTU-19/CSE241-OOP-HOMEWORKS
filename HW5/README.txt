The game has a menu which has 7 option:
1. option:The user can play just a game in one HexVector object.if the game is ended,the game is restarted for new game.if it is not,the game continues from last movement.

2. option:The user can play just a game in one HexArray1D object.if the game is ended,the game is restarted for new game.if it is not,the game continues from last movement.

3. option:The user can play just a game in one AdapterHex object.if the game is ended,the game is restarted for new game.if it is not,the game continues from last movement.

4. option:The user can play more than a game at the same time.Each game takes a move one after another and moves on to the next game.if the all games are ended, the option is ended.if it is not,it continues from last movement for each game.

5. option:The user can compare two games in terms of gameboard. 

6. option:The user can control the games for the board and the moves.

7. option:The user can exit the game.


Also the user can save it's game and load the game another time.
Also the user can use 'RESET' command for resetting the gameboard.
Also the user can use 'LAST MOVE' command for reaching the last move.
Also the user can use 'SET SIZE' command for resetting the board size.
